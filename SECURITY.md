# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in the AGLedger MCP Server, please report it responsibly.

**Email:** security@agledger.ai

We will acknowledge your report within 48 hours and aim to provide a fix within 90 days.

Please do not open a public GitHub issue for security vulnerabilities.

## Security Practices

- API keys are never logged or included in error messages
- All API communication uses HTTPS
- No secrets are stored in source code
- Dependencies are minimal and actively maintained
